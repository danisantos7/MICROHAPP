# **App Name**: MICROHAPP

## Core Features:

- Risk Input Form: Collect user data: sex (dropdown), age (number input), and packs of cigarettes smoked per year (number input).
- Risk Calculation: Calculate the probability of tumor based on the formula using the data introduced by the user.
- Risk Assessment Popup: Display a popup based on calculated risk. Show a popup saying X tests must be done if risk > +7% and different tests if the risk is <-7%.

## Style Guidelines:

- Primary color: Deep blue (#293B5F) for a sense of trust and health awareness.
- Background color: Light blue-gray (#D4E0E9) for a clean, clinical feel.
- Accent color: Soft teal (#39A9CB) to draw attention to CTAs, while maintaining a medical vibe.
- Additional color: Blood red (#8B0000) sparingly used for highlighting critical risk indicators.
- Body and headline font: 'Inter', a sans-serif, provides a clean and modern reading experience.
- Simple single page design. App title on the header, 3 form questions on the middle of the screen. Result as a popup. Footer stating 'Desarrollado por Daniel Santos Olmo Montoya'.
- Subtle animations to indicate calculation and to draw attention to popup results.
